Dreamweaver

¿Qué es Dreamweaver?

Adobe Dreamweaver es un programa para diseñar sitios web (paginas webs) desarrollado por Adobe Inc. Dreamweaver es un editor visual de páginas web que le permite crear, editar y administrar sitios web y páginas web sin tener que aprender HTML.

Dreamweaver fue creado por Macromedia en 1997. Adobe compró Macromedia en 2005 y continuó con el desarrollo del programa. Dreamweaver es parte del conjunto de aplicaciones de Adobe Creative Cloud.

Principales Caracteristicas

Algunas de las funciones de Dreamweaver son:
  • Funciona en Windows, Mac OS y permite diseñar tanto visualmente como mediante código.
  • Compatible con varios lenguajes de programación, incluidos HTML, JSP, CSS, ASP, PHP, XML y muchos más.
  • Ofrece una paleta de diseño con una gran cantidad de opciones de color.
  • Proporciona una herramienta de publicación que lo ayuda a administrar fácilmente sus archivos, tanto local y online.
  • Edite texto e imágenes para sus páginas web.

Costo

Dreamweaver tiene un costo de suscripción de $20.99 USD al mes o una suscripción a Creative Cloud por $52.99 USD al mes, que incluye todas las aplicaciones de Adobe en la Suite Creative Cloud.

Dreamweaver proporciona herramientas esenciales para crear sitios web, tanto para principiante y profesionales.
